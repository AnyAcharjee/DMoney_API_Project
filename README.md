# 🚀 DMoney API Testing Project

## 📌 Project Description
This is a basic API testing project for a Mobile Financial Service (MFS) system.  
It demonstrates API testing using Postman and Newman, including automated test execution and report generation.

---

## 🛠️ Technology Used
- Node.js  
- Postman  
- Newman  
- HTML Extra Report  

---

## ▶️ How to Run

### 🔹 Step 1: Initialize Project
`npm init -y`


### 🔹 Step 2: Install Dependencies
`npm install newman newman-reporter-html`
### 🔹 Step 3: Run Test Collection
`newman run collection.json`
### 🔹 Step 4: Generate HTML Report
`node .\report.js`
